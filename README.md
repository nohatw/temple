01. 先安裝 [Homebrew](https://brew.sh/){:target="_blank" rel="noopener"}
打開 Terminal ，輸入下面指令。
```

```
02. 安裝 Homebrew 並執行下列命令，就能安裝好 Git：
`brew install git`
